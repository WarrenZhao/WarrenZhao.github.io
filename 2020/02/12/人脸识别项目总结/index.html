项目背景

人脸识别是当前AI应用落地的方向之一，人脸识别项目一般有两个技术点：第一个点是在一张图片中检测出人脸部分，标出人脸的坐标，找出人脸的关键点（landmarks，比较简单的是五个关键点，双眼、鼻头和嘴巴左右两个点）；第二个技术点是将检测出来的人脸进行识别，然后将识别出来的具体信息再进行工程处理，给出识别结果达到识别目的。

在人脸检测部分，本文参考的主要文献是2016年发表在IEEE Single Processing Letters期刊上的Joint Face Detection and Alignment using Multi-task Cascaded Convolutional Networks文章，该期刊在2019年CCF（中国计算机学会）推荐的期刊目录中属于是C类期刊，但是影响因子在逐年增高。如下图所示:

当然，论文提出的MTCNN网络效果还是很好的，有如下亮点：

  • 深度卷积神经网络
  • 层叠网络多任务
  • 又粗到细逐步定位物体

技术点一：人脸检测

Joint Face Detection and Alignment using Multi-task Cascaded Convolutional Networks这篇文章将人脸检测和人脸对齐两个任务合并在了一起，这篇文章提出了一个基于深度学习的three-stage（三阶段）深度学习网络，同时还提出了一个在线困难样本挖掘策略（只使用前向传播loss由大到小前70%的数据进行反向传播）进行样本选择。使用的数据集有：

  • FDDB: Face Detection Data Set and Benchmark
  • WIDER FACE: 这是一个用于人脸检测的数据集，具有61个场景，收集了32203张图片，拥有393703条人脸标注数据，标注格式使用的是txt或mat格式，每张图片有三部分标注，第一行是文件相对路径；第二行是本张图片的人脸个数（比如第二行如果是3的话，代表本图标注了3张人脸，并且接下来每一行代表一张人脸的标注信息）；接下来的每一行代表一张人脸的标注信息，没个信息要素用空格隔开，分别是：左上角点x，左上角点y，bounding box的宽，bounding box的高，是否模糊，表情是否夸张，光照条件，遮挡情况，姿势，是否有效。如果做人脸检测，用这个数据训练就足够了。
  • AFLW: Annotation Facial Landmarks in the Wild

在此之间的工作，文章主要提到了：1.希尔特征提取+Adaboost分类器，无法适用真实世界的应用场景；2.DPM(deformable part models)，需要大量的计算开销和海量标注；3.基于CNN的方法，模型结构复杂，并且忽略了bound-box和landmarks的内在关联。

关于单独的人脸对齐，有基于回归的方法和基于模板拟合的方法。

这篇文章的作者把文章的主要贡献总结为：

  1. 提出了一个级联的、轻量级的CNN架构，并且具有实时监测功能；
  2. 提出了一个有效的方法进行困难样本在线选择，用于提高模型的表现；
  3. （空话）为了展示模型的效果和现有的方法的对比，进行了广泛实验。

算法框架

  1. resize->image pyramid图像金字塔；
  2. P-Net（Proposal Net）->候选框窗口和bounding box的回归向量->NMS（非极大值抑制）去除重叠过多的候选框；
  3. R-Net（Refine Net）->bounding box的回归操作和NMS进一步去除多余的候选框；
  4. O-Net（Output Net）->与R-Net功能类似，但是输出的细节更多，包括bounding box和脸部的landmarks坐标；

CNN架构

使用了3*3的卷积核同时增加了深度（我猜测应给是通道数）来获得更多的细节和更好的表现。对比的文献是2015年CVPR的文章A convolutional neural network cascade for face detection。

训练

训练过程中，拆分出了三个子任务。第一个任务是是一个二分类问题，目标是判断是否是人脸，损失函数使用的是交叉熵。

第二个子任务是找出bounding box的位置，是一个回归问题，采用的是欧式距离损失函数，采用的点是矩形框左上角点的坐标+矩形框的宽和高，因此有四个值。

第三个子任务也是一个回归问题，采用的依旧是欧式距离损失函数，点坐标采用的是landmarks的五个点的坐标，因此有10个值。

对于文章中设计出来的三个网络，不同网络的主要任务不同，因此网络中的权重是不同的，对于P-Net和R-Net，由于更加注重任务一和任务二，因此$\alpha_i$有所不同，$\alpha_{det}=1$, $\alpha_{box}=0.5$, $\alpha_{landmark}=0.5$。在O-Net中，由于网络需要输出landmarks$\alpha_{det}=1$, $\alpha_{box}=0.5$, $\alpha_{landmark}=1$。而$\beta_i^j$是样本指示器取值为0或者1，当判断为非人脸时，box和landmark取值为0，判断为人脸时box和landmark取值为1。

对于数据的选取，每一个mini-batch都输入网络进行前向传播（由input到output进行计算），然后只取loss由大到小前70%的数据作为hard sample，进行反向传播操作。这样就完成了online hard sample mining。根据作者的说法，实验结果有显著的提升。

技术点二：人脸识别

这部分的技术点主要参考了facenet的论文，论文中提到，facenet并不是一种网络结构，而是一个系统的名称。文章的主要贡献在于：

  1. 提出了一个一个新的损失函数，用于人脸验证（是否是同一个人）、人脸识别（该人是谁）和人脸聚类（给出的一堆人脸中那些是同一个人）。
  2. 一种embedding的思路，提取图片中人脸的特征并映射到一个向量中，这个向量可以很稀疏（128维）。
